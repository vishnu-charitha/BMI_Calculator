# ⚕️ BMI Calculator

A simple and interactive **BMI (Body Mass Index) Calculator** built using **Python** and **Streamlit**.

This web app allows users to calculate their BMI based on weight and height, classify health status, and receive basic health suggestions.

## 🚀 Features

* Calculate BMI instantly
* User-friendly interface
* BMI category classification
* Health suggestions based on BMI
* Input validation
* Responsive web app using Streamlit

## 🛠️ Tech Stack

* Python
* Streamlit

## 📌 BMI Formula

BMI = Weight (kg) / Height² (m²)

## ▶️ Run Locally

1. Clone the repository

```bash
git clone https://github.com/vishnu-charitha/BMI_Calculator.git
```

2. Navigate to project folder

```bash
cd BMI_Calculator
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the app

```bash
python -m streamlit run app.py
```
